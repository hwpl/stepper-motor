Detailed Description

Driver for an 8-step stepper motor

Example

using Stepper = StepperMotor::Driver;
Stepper motor({2, 3, 4, 5});
void loop() {
motor.turn(-90);
delay(1000);
motor.turnTo(0);
delay(3000);
}
